# 206 递归论（Recursion Theory）

## 1. 概述

递归论，又称可计算性理论，是数理逻辑和理论计算机科学的重要分支，研究可计算函数、算法的本质及其极限。递归论为理解算法、复杂性和不可判定性等问题提供了理论基础。

## 2. 主要分支与核心问题

- 原始递归函数与μ递归函数
- 图灵可计算性与λ演算
- 不可判定性与停机问题
- 递归可枚举集与递归集
- 归约与复杂性

## 3. 理论体系与形式化表达

- 可计算函数的定义：
  - 原始递归函数、μ递归函数的递归方程
- 图灵机模型与λ演算
- 不可判定性证明（如停机问题）：
  \[
  \text{不存在通用算法判定任意程序是否停机}
  \]

## 4. Haskell实现示例

```haskell
-- 原始递归函数示例：阶乘
fac :: Integer -> Integer
fac 0 = 1
fac n = n * fac (n - 1)

-- μ递归函数示例：寻找最小满足条件的自然数
mu :: (Integer -> Bool) -> Integer
mu p = head [n | n <- [0..], p n]
```
