## 1. 内容简介

本层级聚焦于数学基础、形式语言、逻辑系统、类型理论、集合论、范畴论、模型论、证明论、递归论、信息论、自动机理论等内容，强调学术性、结构化与多表征表达，兼顾Haskell、Rust、Lean等多语言对比。
