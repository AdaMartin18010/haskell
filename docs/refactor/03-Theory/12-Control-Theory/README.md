# 控制理论 (Control Theory)

## 概述

控制理论是研究系统动态行为和控制策略的数学理论，是现代工程和科学的重要基础。本目录包含控制理论的核心内容，从线性控制到非线性控制，从经典控制到现代控制，从理论分析到实际应用。
