# 自动机理论 - 主索引

## 📋 概述

自动机理论是计算机科学的基础理论之一，研究抽象计算模型和形式语言。它提供了理解计算本质、语言识别和模式匹配的数学框架。

## 🏗️ 理论架构

### 核心概念

- **有限自动机**：最简单的自动机模型，用于正则语言识别
- **下推自动机**：带栈的自动机，用于上下文无关语言
- **图灵机**：通用计算模型，定义了可计算性
- **形式语言**：由语法规则定义的字符串集合
- **计算复杂性**：算法和问题的复杂度分析

### 理论层次

1. **有限自动机**：正则语言和状态机理论
2. **上下文无关语言**：下推自动机和语法分析
3. **图灵机理论**：可计算性和复杂性理论
4. **形式语言理论**：语言层次和语法理论
