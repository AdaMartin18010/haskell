## 概述

本目录系统性整理Haskell的控制流机制，涵盖条件表达式、递归、模式匹配与高阶函数等主题，所有内容均配备严格的数学定义、Haskell代码示例和多表征形式。

- **条件表达式**：if-then-else、guards、case表达式
- **递归函数**：基本递归、尾递归、结构递归
- **高阶函数**：函数作为参数和返回值、组合与管道
